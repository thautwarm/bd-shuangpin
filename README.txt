大词库音调+双拼重码消解+云输入。

单字全码：o+双拼+「音调1234=asdf」+「86五笔第一码」+「最后一笔 横竖撇捺折=ghtyn」
词语全码：双拼+「末字音调1234=asdf」+「首音调1234=asdf」+「倒数第二字音调1234=asdf」

e.g.,

```
即刻=jikeds
最吼=zvhzd
最后，最厚=zvhzf
```
输入法生成:

```
bash cmd.sh
```

需要词库每行格式为：`词`+`\t`+`频率`+`\n`.

建议使用个人搜狗词库。兄弟，非常快乐。

类似的全拼方案对staged-im-gen.py修改即可。

## 关于云输入

有两种，一种基于sqlite3(`sqlserver.py`)，加载很快但是输入会有一点卡顿。

第二种基于trie(`trieserver.py`)，我做了改进，可以klog(n)高效查找词频前N个，太快了太快了（但是每次启动输入服务需要几分钟，建议开机启动，不要将服务与输入法一同关闭）。
